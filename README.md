## How to Run

```bash
chmod +x scripts/*.sh

./scripts/script1_system_info.sh
./scripts/script2_package_inspector.sh
./scripts/script3_disk_auditor.sh
./scripts/script4_log_analyzer.sh /var/log/syslog error

```
